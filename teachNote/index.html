# js 部分

  1. 获取元素的方法
    通过标签名：document.getElementByTagName 返回值为数组
    通过类名：document.getElementByClassName 返回值为数组
    通过 id 名: document.getElementById
    通用：document.querySelector 只能获得到第一个
  2. 绑定事件的方法，以及区别
    Element.onEventName = callback 只会生效最后一个设置的
    Element.addEventLister (eventName, callback) 可以生效多个
  3. 定时器有哪两种
    setTimeout (callback, timeout) 超时后执行 callback
    setInterval（callback, time） 每过 time 毫秒，执行一个 callback
  4. 事件捕获是什么，事件冒泡是什么
    捕获是从外到内触发事件，冒泡是从内到外触发事件，默认为冒泡
  5. 什么是事件对象，事件对象的 target 属性是什么
    事件对象是绑定事件的 callback 的一个参数 event，上面有很多属性，记载了一些信息， target 保存的就是触发事件的元素
  6. 什么是事件委托，应用场景
    事件委托就是本来是想给所有的子节点绑定事件的，但是我们选择委托给父元素，也就是给父元素绑定节点，然后当子节点的相应的事件被触发的时候，冒泡到父元素，从而触发绑定给父元素的事件，可以通过事件对象的 target 属性获取被触发事件的子节点。 -->
